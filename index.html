estudando git e gitHub

estou fazendo um curso de Java e essa é uma seção bonus do curso

alguns codigos do git:

    git init (colocar uma pasta git no seu projeto)
    git add . (para pegar os arquivos e mandar no gitHub)
    git commit -m "Mensagem explicativa"
    git branch -M main
    git remote add origin git@github.com:seuusuario/seurepositorio.git
    git push -u origin main (para enviar seu commits no repositorio que voce colocou)
    git status(para ver as mçodificações etc...)
